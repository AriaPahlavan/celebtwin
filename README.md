# Celebtwin Application

## Development Tools:
* [Git](http://git-scm.com/)
* [React](https://reactjs.org/)
* [Tachyons](https://tachyons.io/)
* [Particlesjs](https://vincentgarreau.com/particles.js/)

## Run Locally:
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
